# KANJI-ETYMOLOGY-DICTIONARY

A chrome extension made for Japanese language learners learning kanji.

## Features

* Search and look up kanji
* Customizable settings
* Tab-based navigation
* Optional hover popup

## Requirements

* Google Chrome

## Running locally

1. Clone the repository:

   ```bash
   git clone https://github.com/reffij/Kanji-Etymology-Dictionary
   ```
2. Open chrome and navigate to ```chrome://extensions```
3. Enable Developer mode using the toggle in the top-right corner.
4. Click **Load unpacked** on the top left
6. Navigate to and select the cloned ```Kanji-Etymology-Dictionary``` project folder
7. (Optional) Pin the extension to the Chrome toolbar for easier access.
